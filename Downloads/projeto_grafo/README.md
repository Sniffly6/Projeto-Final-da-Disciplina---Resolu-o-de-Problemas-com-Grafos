# Projeto Final - Grafos

## Tema

Rede de cidades conectadas por estradas.

Cada cidade é representada por um vértice e cada estrada por uma aresta ponderada pela distância em quilômetros.

## Estrutura do Projeto

```text
dados/
src/
README.md
relatorio.md
```

## Requisitos

Python 3.10 ou superior.

## Como executar

Clone o repositório:

```bash
git clone <url-do-repositorio>
```

Entre na pasta:

```bash
cd projeto-grafos-cidades
```

Execute o programa:

```bash
cd src
python main.py
```

## Funcionalidades

* Construção do grafo a partir de arquivo CSV.
* Busca em largura (BFS).
* Verificação de conectividade entre cidades.
* Caminho mínimo com Dijkstra.
* Árvore Geradora Mínima com Kruskal.
